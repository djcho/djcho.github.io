## 🔍문제

길이가 n이고, "수박수박수박수...."와 같은 패턴을 유지하는 문자열을 리턴하는 함수, solution을 완성하세요. 예를들어 n이 4이면 "수박수박"을 리턴하고 3이라면 "수박수"를 리턴하면 됩니다.

##### 제한 조건

- n은 길이 10,000이하인 자연수입니다.

##### 입출력 예

| n    | return     |
| ---- | ---------- |
| 3    | "수박수"   |
| 4    | "수박수박" |



## 📝내 풀이

### 코드

```c++
#include <string>
#include <vector>

using namespace std;

string solution(int n) {
    string answer = "";
    string str;
    for(int i = 1; i <= n; i++){
        str = i%2 ? "수" : "박";
        answer.append(str);
    }
    return answer;
}
```
